Suspense#

This is a wrapper around React.Suspense, but better. Hire me Facebook.

Props:

  • fallback: React.ReactNode, the element to be shown while loading.
  • errorFallback: React.ReactNode, the element to be shown if an error occurs (can't be used with noErrorBoundary).
  • noErrorBoundary?: true (optional), if present, disables catching errors for this element.

Example:

import { Suspense } from "./Core/Core.tsx";
// ...
<Suspense fallback={<Spinner />}>  <Element /></Suspense>;

ErrorBoundary#

Catches errors and logs them to the console, and renders a fallback.

Props:

  • fallback: React.ReactNode, the element to be shown if there's an error.

Example:

import { ErrorBoundary } from "./Core/Core.tsx";
// ...
<ErrorBoundary fallback={<ErrorMessage />}>  <Element /></ErrorBoundary>;
