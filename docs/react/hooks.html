Hooks

The official React docs about hooks are here. You know you can Google these, right?

Of course because my old friend Mark Zuckersdottir at Facebook hasn't gotten around to it yet, I've had to make some custom hooks. Here they are.

useStartLoading and useFinishLoading#

Can you guess what these do? I'll give you 30 seconds. Wrong. These hooks control the loading bar that shows up at the top of the screen. It's rare to have to use these, although you do want to use them if you manually redirect a user. For example,

const mutation = graphql`    mutation ComponentMutation {        componentMutation    }`;export default function Component() {  const [commit, _isInFlight] = Relay.useMutation(mutation);  const navigate = ReactRouter.useNavigate();
  const onSubmit = function () {    const onCompleted = function (data: unknown) {      Console.log(data);      navigate("/", {        state: { redirected: true },      });    };
    const onError = function (error: Error) {      Console.error(error);      useFinishLoading();    };
    useStartLoading();    commit({      variables: {},      onCompleted: onCompleted,      onError: onError,    });  };
  const element: React.ReactElement = <></>;  return element;}
info

You have to pass in { state: { redirected: true } } to navigate(). You HAVE to. Because I said so. Otherwise it doesn't work.

useToken#

Arguments:

  • value?: string | null (optional), the value of the token.

Returns: the token, or undefined if it doesn't exist.

If value isn't provided, the value of the token is returned. If it's a string, then the token is set to value. If it's null, the token is cleared (used to log out a user).

useRefresh#

Arguments:

  • fetchRefresh: () => Promise<unknown>, the function used to fetch the refresh token.

Returns: a pair containing a function to indicate loading status, and an effect function to fetch the token on refresh.

This is literally ever only used once, so if you're using this you tripping. Example:

export default function App() {  /* Obtains refresh JWT. */  const [loading, effect] = useRefresh(Client.fetchRefresh);  React.useEffect(effect, []);  if (loading()) {    return <Spinner />;  } else {    return <Content />;  }}
